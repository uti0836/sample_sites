# 化学
## ① 中和滴定
中和滴定のシミュレーションを行うWebアプリです。

[中和滴定 :fontawesome-brands-windows:](https://okiraku-lab.github.io/neutralization_titration/neutralization_titration.html){:target="_blank" .md-button }
![](./image/中和滴定_1.png)
!!!note "アプリの使い方"
	- 滴下ボタン（💧）を押せば、ビーカー内に溶液が1滴、滴下されます
	- 滴下する溶液の種類、モル濃度、滴下量は、左下のラジオボタンからそれぞれ選ぶことができます（塩基の水溶液は、水酸化ナトリウム水溶液で固定です）
    - 🔁マークを押せば、滴定の方向性（酸aq → 塩基aq、塩基aq → 酸aq）が切り替わります
	- 滴定曲線の変色域をクリックすれば、指示薬の種類が切り替わります
