# 化学
## ① 化学式を適切な種類に分類する
化学式を「単体」と「化合物」に分類する場合、**式中のアルファベットが大文字1つで単体**、**大文字2つ以上で化合物**です。たとえば、Cl₂はアルファベットの大文字（元素）が1つなので、単体であり、H₂Oはアルファベットの大文字が2つなので、化合物です。**式の右上に電荷を表す数字があれば、単原子イオンもしくは多原子イオン**となります。このように、化学式の種類は文字列のパターンから見分けることができます。

化学式を見分けるだけでは、NO₃⁻などの粒子の構造をイメージすることはできませんが、このような知識を1つずつ積み上げることで、いずれ高度な概念を習得することができます。

[化学式を適切な種類に分類する :fontawesome-brands-windows:](https://okiraku-lab.github.io/categorize_chemicalformula/categorize_chemicalformula.html){:target="_blank" .md-button }
![](./image/化学式の分類_1.png)
!!!note "アプリの使い方"
	- 化学式をドラッグし、「単体」、「化合物」、「単原子イオン」、「多原子イオン」というラベルが書かれたビーカーにドロップしてください
	- すべての化学式をビーカーにドロップすれば、分類が正しい場合、正解のエフェクトが表示されます
	- 正解エフェクトの後、10秒以上経過した後、画面をクリックすれば、ゲームを再開できます

## ② 組成式をつくる
組成というのは**比**のことであり、組成式というのは**原子の数の比を使って、物質を表した化学式**です。イオン結晶や共有結合の結晶、金属結晶などを表す際、組成式が使われます。たとえば、NaClはNa⁺とCl⁻が1：1の数の比で結びついたものです。また、Ca(OH)₂はCa²⁺とOH⁻が1：2の数の比で結びついたものです。OH⁻のような多原子イオンの右下に2以上の数字が来る場合、イオン式全体をカッコで囲みます。

[^note1] 陽イオンと陰イオンの組み合わせから、どのような組成式ができるか、考えてみましょう。
[^note1]: 組み合わせの関係上、組成式が存在しない場合、分子式が描かれます。

[組成式をつくる :fontawesome-brands-windows:](https://okiraku-lab.github.io/compositional_formula/compositional_formula.html){:target="_blank" .md-button }
![](./image/組成式_1.png)
!!!note "アプリの使い方"
	- 陽イオンと陰イオンを1つずつ選択すれば、その組み合わせに対応する組成式が描かれます
	- 組み合わせの関係上、組成式が存在しない場合は、分子式が描かれます（H₂OやH₂SO₄など）

## ③ 電子式をつくる
電子式というのは、**原子どうしの共有結合の様子を示すために、元素記号の周りに電子を点（・）で表した化学式**です。共有結合には、それぞれの原子が電子を1つずつ共有する単結合、2つずつ共有する二重結合、3つずつ共有する三重結合があります。**1組の共有結合を1本の線（－）で置き換えた化学式を、構造式**といいます。例外はありますが、電子式や構造式は分子を表すために使われることが多いです。

平面では、分子の立体構造を表現することは困難ですが、分子の形が鎖状に伸びたものであれば、このWebアプリで電子式をつくることができます。まずは、水（H₂O）、二酸化炭素（CO₂）、アンモニア（NH₃）、メタン（CH₄）など、化学の授業で頻出の分子をつくり、その後、たとえば酢酸エチルなど、複雑な分子の構造式をつくることにチャレンジしてください。

[電子式をつくる :fontawesome-brands-windows:](https://okiraku-lab.github.io/electronic_formula/electronic_formula.html){:target="_blank" .md-button }
![](./image/電子式_1.png)
!!!note "アプリの使い方（PC推奨）"
	- 周期表上の元素記号をクリックすれば、その原子の電子式が現れます
	- 電子式やその周りの電子は、ドラッグで動かすことができます
	- 各原子が電子を共有すれば、共有結合（単結合、二重結合、三重結合など）ができます
	- 🔍マークを押せば、検索モードになります。電子式をクリックすれば、物質の情報が表示されます
	- 🔁マークを押せば、電子式と構造式が切り替わります
