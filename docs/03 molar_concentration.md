# 化学
## ① 質量パーセント濃度とモル濃度
**質量パーセント濃度**とは、溶液に含まれる溶質の質量の割合を百分率で表した濃度です。また、**モル濃度**とは、溶液1L当たりに含まれる溶質を**物質量（mol）**で表した濃度です。この2つの濃度のうち、化学ではモル濃度の方をよく使います。化学の実験では、何らかの物質が溶けている溶液をはかり取り、ビーカーなどの容器に注ぐという操作をたびたび行います。モル濃度がわかれば、その溶液に何mol（→ 何g）の溶質が溶けているか、簡単に求めることができます。

質量パーセント濃度とモル濃度の換算は、計算に慣れるまでは、少し難しく感じるかもしれません。この2つの濃度を相互に換算するためには、**溶液の密度**の情報が必要です。

水の密度は約1.0g/mLですが、水に何らかの物質が溶けている場合、密度は水と同じとはいきません。たとえば、水に塩化ナトリウム（NaCl）を溶かすと、溶質である塩化ナトリウムは、水中でNa⁺とCl⁻に電離して、水分子の間に入り込みます。これらのイオンは水分子より重く、水溶液の体積が変わらなくても、全体として質量が増えることになります。

質量パーセント濃度を求めるためには、溶質と溶液の質量（g）を求める必要があります。モル濃度では溶液は体積（L）ベースで表されているため、2つの濃度の換算には溶液の密度の情報が不可欠というわけです。

## ② 特定のモル濃度の溶液を調製する
特定のモル濃度の溶液の調製のしかたを理解することは重要です。たとえば、0.10mol/Lの塩化ナトリウム水溶液を500mLつくる場合を考えましょう。溶液の調製では、ビーカーやメスフラスコなどの器具を使います。この操作をマスターすれば、溶質粒子がビーカーからメスフラスコにどのように受け渡されるか、具体的にイメージできるようになります。

溶液の調製では、操作を覚えるだけではなく、塩化ナトリウムが何g必要か計算する必要もあります。**「モル = モル濃度 × 体積」という式を使い、溶質の質量を求める計算は、その後のすべての定量的な考え方の土台となります**。
